## Клонирование репозитория
Склонируйте репозиторий с исходным кодом и тестами:
```bash
git clone https://github.com/dima5778/konfig2
cd konfig2
```

## Запуск
Запуск эмулятора
```bash
python src/shell_emulator.py
```

## Структура проекта
